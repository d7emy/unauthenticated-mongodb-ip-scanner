# unauthenticated-mongodb-ip-scanner

Scan all given ips in [ips.txt] file for an unAuthinticated Mongo DATABASE.

## run Command

- to run it you must already installed [go 1.15] or later (this project run stable at 1.19.0).

```bash
    go run .
```
